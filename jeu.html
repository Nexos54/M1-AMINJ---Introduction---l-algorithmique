Remarques pour le projet

Vous avez là une interface très sommaire qui devrait vous fournir de quoi avoir une base pour utiliser le javascript.
Le bouton "nouvelle partie" ne fait que rafraichir la page.
Le bouton "proposer" permet de récupérer les valeurs mises dans les 2 champs.
A but d'expliquer comment manipuler champs input et les paragraphes, des exemples de manipulation sont inscrits dans le code javascript et je vous recommande :

  • d'analyser celui-ci en regardant le contenu du fichier
  • de tester différentes choses pour bien prendre en main la gestion du HTML par le javascript
  • débugger et comprendre les erreurs que vous pourriez générées sur celui-ci

Ce fichier restera disponible sur la page du projet, vous pouvez donc tester sans crainte et à loisir.


La casse des méthodes ou propriétés en javascript est importante ! Une méthode toto sera différente d'une méthode Toto ou toTO.

La méthode document.getElementById(id_de_l_element) permet de récupérer l'élément qui a l'ID id_de_l_element dans la page.
En affectant l'élément récupéré à une variable (ou une constante), il est assez simple de manipuler l'élément.
Vous trouverez des exemples dans le code javascript.

La propriété innerHTML permet de récupérer ou de changer le contenu d'une balise paragraphe.

La propriété value permet de récupérer ou de changer le contenu d'un champ de texte.
